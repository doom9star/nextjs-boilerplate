# NEXTJS BOILERPLATE

## INSTRUCTIONS

- clone repository
  - git clone https://github.com/doom9star/nextjs-boilerplate.git
- run setup
  - ./scripts/setup.sh
- fill environment variables
  - zeus/.env
  - apollo/.env
- run whenever you modify **zeus/schema.prisma**
  - ./scripts/migrate.sh
- start tasks
  - ctrl + shift + p -> Tasks: Run Task -> setup
- stop tasks
  - ctrl + shift + p -> Tasks: Terminate Task -> setup
